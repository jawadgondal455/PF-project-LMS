##  Features

- **User Authentication**
  - Sign Up
  - Sign In
- **Book Management**
  - Add Book
  - View Books
  - Update Book Quantity
  - Delete Book
- **Book Transactions**
  - Issue Book
  - Return Book
- Logout functionality

##  Files and Structure

| File Name        | Description                                  |
|-----------------|----------------------------------------------|
| `project_1.cpp` | Contains full program including menu, functions, and logic |
| `users.txt`     | Stores registered users (username & password) |
| `books.txt`     | Stores books information (ID, Name, Quantity) |

##  How to Compile and Run

Open the project folder in VS Code

Open Terminal

project_1
