##  Features

- **User Authentication**
  - Sign Up
  - Sign In
- **Book Management**
  - Add Book
  - View Books
  - Update Book Quantity
  - Delete Book
- **Book Transactions**
  - Issue Book
  - Return Book
- Logout functionality

## How to Compile and Run

Follow these steps to compile and run the project:

1. **Open the Project Folder**  
   Open the folder containing the project files in **Visual Studio Code**.

2. **Open the Terminal**  
   Go to **Terminal → New Terminal** in VS Code, or press ``Ctrl + ` `` to open the terminal.

3. **Compile the Program**  
   Type the following command and press **Enter**:
   ```bash
   g++ project_1.cpp -o project_1
This will compile project_1.cpp and generate an executable file named project_1.

# Run the Program
After compilation, run the program using the command\
./project_1
